## **Installation**

1. **Clone the repository:**

   ```bash
   git clone https://github.com/your-username/GameHub.git
   cd GameHub
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```

3. **Start the development server:**

   ```bash
   npm run dev
   ```

4. **Open your browser and go to** `http://localhost:3000`.
